# Skill: Product Requirements

Use this skill to define product requirements for AuthBridge.

## Purpose

Turn product ideas into clear, focused requirements.

## Scope

Check:

- Product goal
- User problem
- Target user
- MVP scope
- Out of scope items
- Product value
- Risks
- Success criteria

## Expected Output

1. Product Goal
2. User Problem
3. Target User
4. MVP Scope
5. Out of Scope
6. Success Criteria
7. Risks
8. Recommendation
