# Skill: Frontend Review

Use this skill to review the AuthBridge frontend.

## Review Scope

Check:

- Registration form layout
- Required fields
- Email validation
- Password validation
- Submit button state
- Loading state
- Success message
- Error message
- Show/hide password behaviour
- Mobile responsiveness
- Accessibility basics
- API URL handling
