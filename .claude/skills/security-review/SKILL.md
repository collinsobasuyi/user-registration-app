# Skill: Security Review

Use this skill to review AuthBridge for basic application security.

## Review Scope

Check:

- Password hashing
- Password not returned in responses
- Password not logged
- `.env` ignored by Git
- `DATABASE_URL` not hardcoded
- CORS is safe for production
- Error messages are safe
- Public user endpoints are marked learning-only
- Email uniqueness is enforced
- Backend does not expose unnecessary sensitive fields
