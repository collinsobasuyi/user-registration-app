# Skill: Architecture Review

Use this skill to review AuthBridge architecture.

## Review Scope

Check:

- Frontend/backend separation
- API boundaries
- Data flow
- Database access
- Environment variables
- CORS
- Deployment model
- Maintainability
- Scalability

## Expected Output

1. Architecture Summary
2. Data Flow
3. Integration Points
4. Strengths
5. Risks
6. Recommended Improvements
7. Architecture Verdict
