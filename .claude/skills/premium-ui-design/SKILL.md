# Skill: Premium UI Design

Use this skill when reviewing or improving the visual design of AuthBridge screens.

## Purpose

Review and improve the premium frontend design of AuthBridge, including registration, login, user dashboard, and admin dashboard screens.

## Review Scope

Check:

- Visual hierarchy
- Layout quality
- Spacing consistency
- Typography
- Colour usage
- Component design
- Form design
- Button styling
- Card design
- Empty states
- Loading states
- Error states
- Success states
- Mobile responsiveness
- Dashboard scalability
- Premium product feel
- Accessibility basics

## Screens Covered

- Registration page
- Login page
- User dashboard
- Admin dashboard
- User management screen
- Profile/account screen
- Error/success screens
