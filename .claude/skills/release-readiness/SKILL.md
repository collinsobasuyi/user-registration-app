# Skill: Release Readiness

Use this skill before saying AuthBridge is ready for demo or deployment.

## Review Scope

Check:

- Product scope is clear
- User story is complete
- Acceptance criteria are met
- UX is usable
- UI feels polished
- Architecture is sound
- Backend works
- Frontend works
- API calls work
- Database saves correctly
- Passwords are hashed
- Secrets are protected
- Deployment config is ready
- README is updated
